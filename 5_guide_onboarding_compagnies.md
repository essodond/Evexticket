## ÉTAPE 1 — INSCRIPTION ET VÉRIFICATION DE VOTRE COMPTE

### 1.1 Créer votre compte compagnie

**Méthode A — Inscription en ligne (recommandée)**

1. Rendez-vous sur **www.evexticket.com/compagnies/inscription**
2. Cliquez sur **"Inscrire ma compagnie"**
3. Remplissez le formulaire avec :
   - Nom officiel de la compagnie (identique au RCCM)
   - Votre nom et prénom (responsable du compte)
   - Numéro de téléphone professionnel
   - Adresse email professionnelle
   - Ville principale d'opération
4. Créez un mot de passe sécurisé (minimum 8 caractères, incluant majuscule, chiffre et caractère spécial)
5. Acceptez les Conditions Générales et la Convention de Partenariat
6. Cliquez sur **"Soumettre ma demande"**

**Méthode B — Inscription assistée**

Contactez notre équipe commerciale pour être guidé lors de l'inscription :
- En personne à notre bureau de Lomé
- Par WhatsApp au +228 XX XX XX XX
- Par email à commercial@evexticket.com

### 1.2 Documents à préparer

Avant de commencer l'inscription, préparez les documents suivants en **format numérique** (photo ou scan) :

| Document | Format accepté | Obligatoire |
|---------|---------------|------------|
| Registre de commerce (RCCM) | PDF ou JPEG | ✅ Oui |
| Autorisation de transport | PDF ou JPEG | ✅ Oui |
| Pièce d'identité du responsable | JPEG | ✅ Oui |
| Logo de la compagnie | PNG ou JPEG (fond transparent idéal) | ✅ Oui |
| Attestation d'assurance RC | PDF ou JPEG | ✅ Oui |
| Numéro de compte pour virements | — | ✅ Oui |
| Photos de vos véhicules | JPEG | Non (recommandé) |

**Conseil :** Prenez des photos claires, avec bonne luminosité. Les documents flous peuvent retarder la validation.

### 1.3 Processus de vérification

Après soumission de votre dossier, voici ce qui se passe :

```
Votre dossier soumis
        │
        ▼
Examen par l'équipe Evexticket (24-48h ouvrables)
        │
        ├── Dossier complet et valide
        │           │
        │           ▼
        │   Email + SMS de confirmation
        │   avec vos identifiants
        │
        └── Dossier incomplet
                    │
                    ▼
            Email vous demandant
            les documents manquants
```

**Délai de vérification : 24 à 48 heures ouvrables**

### 1.4 Première connexion

Après approbation, vous recevez un email avec :
- Votre identifiant de connexion
- Un lien pour définir votre mot de passe
- Le lien de connexion au tableau de bord

**Accès tableau de bord compagnie :** https://app.evexticket.com/compagnie

## ÉTAPE 2 — CONFIGURATION DE VOTRE PROFIL COMPAGNIE

Un profil complet et attrayant inspire confiance aux voyageurs. Prenez le temps de bien le configurer.

### 2.1 Accéder aux paramètres du profil

1. Connectez-vous sur https://app.evexticket.com/compagnie
2. Cliquez sur votre nom en haut à droite
3. Sélectionnez **"Profil de la compagnie"**

### 2.2 Informations de base

| Champ | Description | Exemple |
|-------|-------------|---------|
| Nom de la compagnie | Nom officiel complet | "Compagnie Trans-Togo SARL" |
| Slogan | Phrase courte accrocheure | "Votre confort, notre priorité" |
| Description courte | Max. 150 caractères | "Spécialiste Lomé-Kara depuis 2010" |
| Description longue | Max. 1 000 caractères | Historique, valeurs, services... |
| Numéro de téléphone | Contact voyageurs | +228 90 XX XX XX |
| Email contact | Email visible aux voyageurs | contact@compagnie.tg |
| Adresse gare principale | Adresse complète | Gare routière de Lomé, Bè |
| Site web | Si disponible | www.compagnie.tg |
| Réseaux sociaux | Facebook, Instagram | @compagnieTrans |

### 2.3 Logo et images

**Logo :**
- Format recommandé : PNG avec fond transparent
- Dimensions : minimum 200×200 pixels
- Taille maximum : 2 MB

**Photos de vos véhicules :**
- Ajoutez 2 à 5 photos de haute qualité de votre flotte
- Les voyageurs apprécient voir les bus avant de réserver
- Format : JPEG, minimum 800×600 pixels

**Photo de la gare :**
- Aide les voyageurs à identifier votre point de départ
- Format : JPEG, minimum 800×600 pixels

### 2.4 Coordonnées bancaires pour les virements

**Important :** Sans ces informations, vous ne pourrez pas recevoir vos paiements.

1. Dans le profil, cliquez sur **"Paramètres financiers"**
2. Choisissez votre mode de réception préféré :
   - **Flooz (Moov Africa)** : saisissez votre numéro Flooz
   - **T-Money (Togocel)** : saisissez votre numéro T-Money
   - **Compte bancaire** : saisissez le RIB/numéro de compte
3. Enregistrez et **vérifiez le numéro** avec soin

**⚠️ Attention :** Une erreur dans les coordonnées bancaires peut entraîner une perte de vos paiements. Vérifiez deux fois avant d'enregistrer.

### 3.1 Créer votre premier trajet

1. Dans le tableau de bord, cliquez sur **"Trajets"** dans le menu gauche
2. Cliquez sur **"+ Nouveau trajet"**
3. Remplissez le formulaire (voir Section 3.2)
4. Cliquez sur **"Publier le trajet"**

### 3.4 Configurer des trajets récurrents (ScheduledTrips)

Si vous opérez le même trajet régulièrement (ex : chaque jour), vous pouvez utiliser la fonctionnalité **Trajets Programmés** pour éviter de saisir manuellement chaque départ.

1. Cliquez sur **"Trajets programmés"** dans le menu
2. Cliquez sur **"+ Nouveau programme"**
3. Définissez :
   - La ligne (départ → arrivée)
   - L'heure de départ récurrente
   - La fréquence (quotidien / jours ouvrés / week-end / hebdomadaire)
   - Les jours de la semaine si applicable
   - La date de début et de fin (optionnel)
4. Le système génère automatiquement les trajets correspondants

**Exemple :** Lomé → Kara, tous les jours à 06:00, du 01/07/2025 au 31/12/2025 → 184 trajets créés automatiquement.

## ÉTAPE 4 — GÉRER LES SIÈGES ET LES HORAIRES

### 4.1 Plan de sièges

Pour les bus disposant d'un plan de sièges défini, vous pouvez configurer la disposition dans l'espace **"Gestion des bus"**.

**Types de configurations disponibles :**
- Bus standard 2+2 (ex : 45 places)
- Bus minibus 2+1 (ex : 25 places)
- Bus VIP 2+1 large (ex : 20 places)
- Configuration personnalisée

Une fois le plan configuré, les voyageurs peuvent choisir leur siège au moment de la réservation.

### 4.2 Modifier un trajet existant

Pour modifier un trajet déjà publié :

1. Allez dans **"Trajets"** → **"Mes trajets"**
2. Trouvez le trajet à modifier
3. Cliquez sur l'icône **✏️ (Modifier)**
4. Effectuez vos modifications
5. Cliquez sur **"Enregistrer"**

**⚠️ Important :** 
- Si des réservations existent déjà pour ce trajet, la modification de l'heure ou du prix déclenche automatiquement une notification aux voyageurs concernés.
- Les modifications d'heure ne sont pas autorisées moins de **2 heures** avant le départ si des réservations existent.

### 4.3 Annuler un trajet

En cas de nécessité d'annuler un trajet :

1. Allez dans **"Trajets"** → Trouvez le trajet
2. Cliquez sur **"..." (Options)** → **"Annuler ce trajet"**
3. Saisissez le motif d'annulation (optionnel mais recommandé)
4. Confirmez l'annulation

**Ce qui se passe ensuite :**
- Tous les voyageurs réservés reçoivent immédiatement un SMS et un email
- Les remboursements intégraux sont traités automatiquement sous 24h
- Le trajet disparaît de la plateforme publique

### 4.4 Dupliquer un trajet

Pour créer un trajet similaire rapidement :
1. Trouvez le trajet source
2. Cliquez sur **"..." (Options)** → **"Dupliquer"**
3. Modifiez uniquement la date et/ou l'heure
4. Publiez le nouveau trajet

## ÉTAPE 5 — SUIVI DES RÉSERVATIONS ET DES PASSAGERS

### 5.1 Liste des réservations

La page **"Réservations"** vous donne accès à toutes les réservations de votre compagnie.

**Filtres disponibles :**
- Par date de voyage
- Par trajet (ligne spécifique)
- Par statut (confirmé, annulé, complété, no-show)
- Par passager (recherche par nom)

**Informations visibles pour chaque réservation :**
- Référence de réservation (EVX-XXXXXXXX)
- Nom du passager
- Numéro de téléphone
- Trajet et date
- Siège(s) réservé(s)
- Montant payé
- Statut de la réservation

### 5.2 Liste de passagers par trajet

Pour chaque trajet, vous pouvez obtenir la liste complète des passagers :

1. Allez dans **"Trajets"** → Cliquez sur le trajet concerné
2. Onglet **"Passagers"**
3. Vous voyez la liste avec tous les passagers confirmés

**Télécharger la liste :**
- Cliquez sur **"Télécharger PDF"** → Format imprimable A4
- Cliquez sur **"Télécharger Excel"** → Format tableur pour analyse

**Contenu du document téléchargé :**
- En-tête avec informations du trajet
- Tableau des passagers (N°, Nom, Téléphone, Siège, Statut)
- Résumé (total confirmés, annulés, places libres)

### 5.3 Validation à l'embarquement (Check-in)

Pour valider les billets des voyageurs à l'embarquement, utilisez **l'application mobile Evexticket Compagnie** (disponible sur Android et iOS).

**Processus de validation :**
1. Ouvrez l'application Evexticket Compagnie
2. Connectez-vous avec vos identifiants
3. Sélectionnez le trajet du jour
4. Tapez sur **"Scanner un billet"**
5. Pointez l'appareil photo vers le QR Code du voyageur
6. L'application affiche :
   - ✅ **Billet valide** : nom du passager, siège, confirmation verte
   - ❌ **Billet invalide** : erreur rouge (déjà scanné, faux billet, mauvais trajet)

**Alternative au scan :**  
Si la caméra ne fonctionne pas, le voyageur peut dicter son **code de réservation** (format EVX-XXXXXXXX). Saisissez-le manuellement dans l'application.

### 5.4 Gérer les no-shows et absences

Après le départ, pour les voyageurs qui ne se sont pas présentés :
1. Dans la liste des passagers, cochez les voyageurs non présentés
2. Cliquez sur **"Marquer comme No-Show"**
3. Ces informations sont enregistrées dans votre historique

## ÉTAPE 6 — RAPPORTS, REVENUS ET COMMISSIONS

### 6.1 Tableau de bord financier

Le tableau de bord financier se trouve dans **"Finances"** dans le menu principal.

**Indicateurs affichés :**

| Indicateur | Description |
|-----------|-------------|
| Revenus bruts | Total collecté par Evexticket pour vos billets |
| Commission Evex | Montant déduit par Evexticket (evex_commission %) |
| Revenus nets | Ce que vous recevez après déduction |
| Dernier virement | Date et montant du dernier virement |
| Prochain virement | Date estimée et montant attendu |

### 6.2 Historique des virements

Dans **"Finances"** → **"Virements"** :
- Liste de tous les virements reçus
- Date, montant, mode de paiement
- Statut (traité, en cours)
- Référence de virement

### 6.3 Télécharger les rapports financiers

**Rapport mensuel :**
1. Allez dans **"Finances"** → **"Rapports"**
2. Sélectionnez le mois souhaité
3. Cliquez sur **"Télécharger PDF"** ou **"Télécharger Excel"**

**Contenu du rapport mensuel :**
- Récapitulatif financier (revenus bruts, commissions, nets)
- Liste détaillée de toutes les transactions
- Statistiques par ligne (nombre de billets, revenus)
- Comparaison avec le mois précédent

### 6.4 Comprendre votre commission

Votre taux de commission (`evex_commission`) est défini dans votre convention de partenariat.

**Exemple avec un taux de 8% :**

```
Exemple : 100 billets à 5 000 XOF = 500 000 XOF brut

Revenus bruts     : 500 000 XOF
Commission (8%)   :  40 000 XOF  ← Evexticket
Revenus nets      : 460 000 XOF  ← Votre virement
```

**Réduction automatique selon le volume :**
Si vous vendez plus de 500 billets/mois, votre taux baisse automatiquement au prochain palier. Vous en êtes informé par email.

### 6.5 Statistiques de performance

Dans **"Statistiques"**, accédez aux graphiques suivants :
- 📈 Évolution du nombre de réservations (courbe mensuelle)
- 🗺️ Répartition des réservations par ligne (camembert)
- 📅 Réservations par jour de la semaine (histogramme)
- ⭐ Évolution de votre note moyenne
- 💰 Revenus nets mensuels sur 12 mois

## FOIRE AUX QUESTIONS (FAQ)

### Questions sur l'inscription

**Q1 — Combien de temps prend l'approbation de mon dossier ?**  
R : Généralement 24 à 48 heures ouvrables. Si vous n'avez pas de réponse après 48h, contactez-nous.

**Q2 — Quels documents sont absolument obligatoires ?**  
R : Le RCCM, l'autorisation de transport et la pièce d'identité du responsable sont obligatoires. Le reste peut être fourni après l'approbation initiale.

**Q3 — Ma compagnie n'a pas de RCCM. Peut-elle quand même s'inscrire ?**  
R : Nous vous conseillons de régulariser votre situation légale avant l'inscription. Contactez-nous pour discuter des alternatives.

---

### Questions sur les trajets

**Q4 — Puis-je modifier le prix d'un trajet après publication ?**  
R : Oui, vous pouvez modifier le prix à tout moment. Si des voyageurs ont déjà réservé au prix précédent, ils conservent leur tarif initial. Les nouvelles réservations se feront au nouveau prix.

**Q5 — Combien de trajets puis-je créer ?**  
R : Il n'y a pas de limite au nombre de trajets. Vous pouvez créer autant de trajets que nécessaire.

**Q6 — Puis-je créer des trajets pour des villes non listées ?**  
R : Pour l'instant, seules les 10 villes du réseau Evexticket sont disponibles. Si vous souhaitez ajouter une ville, contactez-nous.

---

### Questions sur les paiements et commissions

**Q7 — Quand vais-je recevoir mon premier paiement ?**  
R : Votre premier virement est effectué le lundi suivant la première semaine complète d'activité, pour toutes les réservations confirmées de cette semaine.

**Q8 — La commission est-elle prélevée sur les billets annulés ?**  
R : Non. La commission n'est prélevée que sur les réservations confirmées et non remboursées. Les annulations avec remboursement n'entraînent pas de commission.

**Q9 — Mon taux de commission peut-il augmenter ?**  
R : Evexticket ne peut modifier votre taux qu'avec un préavis de 60 jours et votre accord. Votre taux initial est garanti pendant la durée de votre convention.

**Q10 — Que se passe-t-il si un voyageur paie mais ne vient pas (no-show) ?**  
R : Vous recevez quand même votre part du paiement (revenus nets après commission). Le no-show est enregistré mais le billet ne peut pas être remboursé selon votre politique.

---

### Questions sur les billets et la validation

**Q11 — Comment fonctionne le QR Code si le voyageur n'a pas de smartphone ?**  
R : Le voyageur reçoit également un code de réservation par SMS. Il peut donner ce code verbal au contrôleur, qui le saisit manuellement dans l'application.

**Q12 — Que faire si l'application de scan ne fonctionne pas le matin du départ ?**  
R : Téléchargez et imprimez la liste des passagers la veille. En cas de problème technique, validez manuellement en vérifiant les noms sur la liste. Signalez le problème au support immédiatement.

**Q13 — Un voyageur peut-il présenter une photo d'écran de son billet ?**  
R : Oui, le QR Code visible sur l'écran du voyageur est identique à celui du PDF et peut être scanné directement.

---

### Questions techniques

**Q14 — L'application fonctionne-t-elle sans internet ?**  
R : L'application de scan nécessite une connexion internet pour valider les QR Codes. Cependant, si la connexion est instable, la liste des passagers peut être téléchargée à l'avance et consultée hors ligne.

**Q15 — Puis-je avoir plusieurs utilisateurs pour mon compte compagnie ?**  
R : Oui, vous pouvez ajouter des membres de votre équipe (responsables de gare, comptables) avec des accès personnalisés. Allez dans **"Équipe"** dans vos paramètres.
